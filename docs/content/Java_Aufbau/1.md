## Einführung

Java ist eine objektorientierte Programmiersprache, die seit 1995 im Einsatz ist. Das Fundament bildet die **Klasse** – ein Bauplan, der **Attribute** (Felder) und **Methoden** (Verhalten) kapselt. Das Verständnis der Klassen‑ und Paket‑Struktur ist entscheidend, um komplexe Anwendungen in überschaubare Bausteine zu zerlegen.

## Java‑Klassen – Grundstruktur

```java
// Datei: src/com/example/Greeter.java
package com.example; // Paketdeklaration

public class Greeter {          // öffentliche Klasse
    private String name;        // privates Feld

    public Greeter(String name) {   // Konstruktor
        this.name = name;
    }

    public void greet() {          // öffentliche Methode
        System.out.println("Hello, " + name + "!");
    }
}
```

* **Package** (`package com.example;`) definiert den Namensraum.
* **Klasse** (`public class Greeter`) ist die zentrale Einheit.
* **Feld** (`private String name`) speichert Daten.
* **Konstruktor** (`Greeter(String name)`) initialisiert die Klasse.
* **Methode** (`public void greet()`) führt Aktionen aus.

---

## Der Einstiegspunkt – `public static void main(String[] args)`

```java
public class Main {
    public static void main(String[] args) {
        Greeter g = new Greeter("Anna");
        g.greet();
    }
}
```

* **`public`**: sichtbar von außen.
* **`static`**: gehört zur Klasse, nicht zur Instanz.
* **`void`**: gibt keinen Wert zurück.
* **`String[] args`**: erlaubt Befehlszeilen‑Parameter.

---

## Sichtbarkeit und Zugriffskontrolle

| Modifikator | Sichtbarkeit | Beispiel |
|---|---|---|
| `public` | überall | `public class Foo` |
| `protected` | innerhalb des Pakets und Sub‑klassen | `protected int count` |
| `private` | nur innerhalb der Klasse | `private String secret` |
| (kein Modifikator) | paket‑privat | `int id` |

!!! info "Access‑Control"
    Gutes Modifikator‑Design verhindert unerwartete Seiteneffekte.

## Kompilieren und Ausführen

```bash
# Im Projekt‑Root
javac src/com/example/*.java
java -cp src com.example.Main
```

* `javac` erstellt `.class`‑Dateien.
* `java` startet die JVM mit dem Klassen­Pfad.

## Befehlszeilen‑Argumente

```java
public class Echo {
    public static void main(String[] args) {
        for (String arg : args) {
            System.out.println(arg);
        }
    }
}
```

Aufruf:

```bash
java Echo Hello World
```

Ausgabe:
```
Hello
World
```
