# Java Aufbau 1 – Paketstruktur & Build

## Lernziele
Nach diesem Kapitel kannst du:
- Eine sinnvolle **Paketstruktur** für ein Java‑Projekt entwerfen.
- Das **Maven‑POM** oder **Gradle‑build** verstehen.
- Klassen in unterschiedlichen Paketen nutzen und importieren.

---

## Einstieg
In Java werden Klassen in Paketen organisiert, um Namenskonflikte zu vermeiden und die Wartbarkeit zu erhöhen.

```
com.example.app            // Basis‑Paket
com.example.app.model      // Datenklassen
com.example.app.service    // Geschäftslogik
com.example.app.controller // REST‑Endpunkte
```
