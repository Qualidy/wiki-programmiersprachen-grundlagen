# XML Grundlagen 1 – Struktur und Syntax

## Lernziele
Nach diesem Kapitel kannst du:
- XML‑Dokumente **glücklich** strukturieren.
- Die wichtigsten **Tags** (`<root>`, `<item>`, `<name>`) benutzen.
- Einfache **Attribute** setzen und auslesen.

---

## Einführung
XML (Extensible Markup Language) ist ein Datenformat, das zur Speicherung und Übertragung von strukturierten Daten verwendet wird.

```xml
<root>
  <item id="1" name="Beispiel" />
</root>
```
