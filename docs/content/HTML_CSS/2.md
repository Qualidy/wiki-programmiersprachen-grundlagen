# HTML Grundlagen 2 – HTML-Elemente

!!! abstract "Lernziele"
    Nach diesem Kapitel kannst du …

    - **Wichtige HTML-Elemente** erkennen und verwenden.
    - **Container-Elemente** (`div`, `span`) korrekt einsetzen.
    - **Medien-Elemente** (`img`, `a`, `video`) verstehen.
    - **Listen-Elemente** (`ul`, `ol`, `li`) anwenden.
    - **Trenn-Elemente** (`hr`, `br`) nutzen.

---

## Container-Elemente

### `<div>` – Block-Element
```html
<div class="container">
  <h1>Überschrift</h1>
  <p>Textinhalt</p>
</div>
```

**Verwendung:**
- Gruppierung von Inhalten
- Layout-Zwecke (z. B. mit CSS Grid/Flexbox)
- Semantische Strukturierung

### `<span>` – Inline-Element
```html
<p>Das ist <span class="highlight">wichtiger Text</span> im Satz.</p>
```

**Verwendung:**
- Text-Auszeichnung
- Styling einzelner Wörter
- Inline-Elemente gruppieren

## Medien-Elemente

### `<img>` – Bild
```html
<img src="bild.jpg" alt="Beschreibung des Bildes" width="300" height="200">
```

**Attribute:**
- `src`: Pfad zum Bild
- `alt`: Alternativtext (Barrierefreiheit!)
- `width`, `height`: Größe (optional)
- `loading="lazy"`: Lazy Loading (Performance)

**Beispiel:**
```html
<img src="logo.png" alt="Firmenlogo" width="150" loading="lazy">
```

### `<a>` – Link
```html
<a href="https://example.com" target="_blank" title="Zur Beispielseite">
  Besuche unsere Website
</a>
```

**Attribute:**
- `href`: Ziel-URL oder Pfad
- `target="_blank"`: Öffnet in neuem Tab
- `title`: Tooltip-Text
- `rel="noopener"`: Sicherheit bei target="_blank"

**Interne Links:**
```html
<a href="ueber-uns.html">Über uns</a>
<a href="#kontakt">Zum Kontakt</a>
```

### `<video>` – Video
```html
<video controls width="640" height="360">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  Ihr Browser unterstützt kein Video.
</video>
```

**Attribute:**
- `controls`: Zeigt Steuerung
- `autoplay`: Auto-Play (mit Muted)
- `loop`: Endlosschleife
- `muted`: Stummgeschaltet

### `<audio>` – Audio
```html
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
  Ihr Browser unterstützt kein Audio.
</audio>
```

## Listen-Elemente

### Ungeordnete Liste (`<ul>`)
```html
<ul>
  <li>Erstes Element</li>
  <li>Zweites Element</li>
  <li>Drittes Element</li>
</ul>
```

### Geordnete Liste (`<ol>`)
```html
<ol>
  <li>Erster Schritt</li>
  <li>Zweiter Schritt</li>
  <li>Dritter Schritt</li>
</ol>
```

**Attribute:**
- `type="1"` (Standard), `type="A"`, `type="a"`, `type="I"`, `type="i"`
- `start="5"`: Start bei 5
- `reversed`: Umgekehrte Reihenfolge

### Beschreibungsliste (`<dl>`)
```html
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>

  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
```

## Trenn-Elemente

### `<hr>` – Horizontale Linie
```html
<p>Erster Absatz</p>
<hr>
<p>Zweiter Absatz</p>
```

**Verwendung:**
- Thematische Trennung
- Visuelle Strukturierung

### `<br>` – Zeilenumbruch
```html
<p>
  Zeile 1<br>
  Zeile 2<br>
  Zeile 3
</p>
```

**Verwendung:**
- Adressen
- Gedichte
- Formatierte Texte

## Semantische Elemente (HTML5)

### `<header>` – Kopfbereich
```html
<header>
  <h1>Meine Website</h1>
  <nav>
    <a href="#start">Start</a>
    <a href="#ueber">Über uns</a>
  </nav>
</header>
```

### `<nav>` – Navigation
```html
<nav>
  <ul>
    <li><a href="#home">Home</a></li>
    <li><a href="#about">About</a></li>
  </ul>
</nav>
```

### `<main>` – Hauptinhalt
```html
<main>
  <article>
    <h2>Artikel</h2>
    <p>Inhalt...</p>
  </article>
</main>
```

### `<article>` – Unabhängiger Inhalt
```html
<article>
  <h2>Blogbeitrag</h2>
  <p>Text...</p>
  <footer>Veröffentlicht am 15.01.2024</footer>
</article>
```

### `<section>` – Inhaltliche Sektion
```html
<section>
  <h2>Kapitel 1</h2>
  <p>Inhalt...</p>
</section>
```

### `<aside>` – Nebeninhalt
```html
<aside>
  <h3>Verwandte Links</h3>
  <ul>
    <li><a href="#">Link 1</a></li>
  </ul>
</aside>
```

### `<footer>` – Fußbereich
```html
<footer>
  <p>&copy; 2024 Meine Website</p>
  <nav>
    <a href="#impressum">Impressum</a>
    <a href="#datenschutz">Datenschutz</a>
  </nav>
</footer>
```

## Formular-Elemente

### `<form>` – Formular
```html
<form action="/submit" method="POST">
  <!-- Formular-Inhalt -->
</form>
```

**Attribute:**
- `action`: Ziel-URL
- `method`: GET oder POST
- `enctype`: Kodierung (multipart/form-data für Dateien)

### `<input>` – Eingabefeld
```html
<input type="text" name="name" placeholder="Name" required>
<input type="email" name="email" placeholder="Email">
<input type="password" name="password" placeholder="Passwort">
<input type="number" name="alter" min="18" max="100">
<input type="date" name="geburtsdatum">
<input type="checkbox" name="newsletter" value="yes">
<input type="radio" name="geschlecht" value="m"> Männlich
<input type="radio" name="geschlecht" value="w"> Weiblich
<input type="file" name="datei" accept=".pdf,.doc">
```

### `<textarea>` – Mehrzeilige Eingabe
```html
<textarea name="nachricht" rows="5" cols="50" placeholder="Ihre Nachricht..."></textarea>
```

### `<select>` – Dropdown
```html
<select name="land">
  <option value="">Bitte wählen</option>
  <option value="de">Deutschland</option>
  <option value="at">Österreich</option>
  <option value="ch">Schweiz</option>
</select>
```

### `<button>` – Button
```html
<button type="submit">Absenden</button>
<button type="reset">Zurücksetzen</button>
<button type="button" onclick="alert('Klick!')">Klick mich</button>
```

### `<label>` – Label
```html
<label for="name">Name:</label>
<input type="text" id="name" name="name">
```

## Tabellen

### `<table>` – Tabelle
```html
<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Alter</th>
      <th>Stadt</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Max</td>
      <td>25</td>
      <td>Berlin</td>
    </tr>
    <tr>
      <td>Anna</td>
      <td>30</td>
      <td>Hamburg</td>
    </tr>
  </tbody>
</table>
```

**Attribute:**
- `border="1"`: Rahmen (veraltet, besser CSS)
- `cellpadding="5"`: Zellenabstand (veraltet, besser CSS)
- `cellspacing="5"`: Zellenabstand (veraltet, besser CSS)

## Praktische Beispiele

### Beispiel 1: Einfache Seite
```html
<!DOCTYPE html>
<html lang="de">
<head>
  <meta charset="UTF-8">
  <title>Beispiel</title>
</head>
<body>
  <header>
    <h1>Willkommen</h1>
  </header>

  <main>
    <article>
      <h2>Überschrift</h2>
      <p>Das ist ein <strong>wichtiger</strong> Absatz.</p>
      <ul>
        <li>Punkt 1</li>
        <li>Punkt 2</li>
      </ul>
    </article>
  </main>

  <footer>
    <p>&copy; 2024</p>
  </footer>
</body>
</html>
```

### Beispiel 2: Kontaktformular
```html
<form action="/kontakt" method="POST">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>

  <label for="nachricht">Nachricht:</label>
  <textarea id="nachricht" name="nachricht" rows="5" required></textarea>

  <button type="submit">Senden</button>
</form>
```

## Häufige Fehler & Best Practices

### Fehler 1: Alt-Text vergessen
```html
<!-- ❌ Falsch -->
<img src="bild.jpg">

<!-- ✅ Gut -->
<img src="bild.jpg" alt="Beschreibung des Bildes">
```

### Fehler 2: Fehlende Labels
```html
<!-- ❌ Falsch -->
<input type="text" name="name">

<!-- ✅ Gut -->
<label for="name">Name:</label>
<input type="text" id="name" name="name">
```

### Fehler 3: Semantische Elemente ignorieren
```html
<!-- ❌ Falsch (nur divs) -->
<div id="header">...</div>
<div id="nav">...</div>

<!-- ✅ Gut (semantisch) -->
<header>...</header>
<nav>...</nav>
```

### Best Practices
1. **Semantische Elemente** verwenden (HTML5)
2. **Alt-Texte** für Bilder immer angeben
3. **Labels** für alle Formular-Elemente
4. **Korrekte Verschachtelung** (keine Block-Elemente in Inline)
5. **Kleine, beschreibende Dateinamen**
6. **UTF-8** als Zeichencodierung

## Zusammenfassung & Checkliste

### Wichtige Konzepte
- ✅ **Container**: `<div>`, `<span>`
- ✅ **Medien**: `<img>`, `<a>`, `<video>`, `<audio>`
- ✅ **Listen**: `<ul>`, `<ol>`, `<dl>`
- ✅ **Trennung**: `<hr>`, `<br>`
- ✅ **Semantik**: `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>`
- ✅ **Formulare**: `<form>`, `<input>`, `<textarea>`, `<select>`, `<button>`, `<label>`
- ✅ **Tabellen**: `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>`, `<td>`

### Checkliste für Code
- [ ] Semantische Elemente verwendet
- [ ] Alt-Texte bei Bildern
- [ ] Labels bei Formularen
- [ ] Korrekte Verschachtelung
- [ ] UTF-8 Zeichencodierung
- [ ] Lang-Attribut bei html
