# Übungsaufgabe: Meine erste persönliche Homepage mit HTML und CSS

## Einleitung

Herzlich willkommen zu Ihrer ersten praktischen Übung im Bereich Webentwicklung! In diesem Kurs haben Sie die Grundlagen von HTML gelernt, der Sprache, die das Rückgrat jeder Webseite bildet. Sie wissen, wie man Inhalte strukturiert, Texte formatiert, Bilder einfügt und Links setzt. Nun ist es an der Zeit, dieses Wissen anzuwenden und Ihre eigene, persönliche Homepage zu erstellen.

Diese Aufgabe soll Sie dazu motivieren, das Gelernte kreativ umzusetzen und gleichzeitig eine nützliche Ressource für sich selbst zu schaffen. Egal, ob Sie Ihre beruflichen Fähigkeiten präsentieren, ein Hobby vorstellen oder einfach nur einen digitalen Raum für Ihre Gedanken gestalten möchten – diese Homepage ist Ihr Projekt!

**Lernziele:**
*   Anwendung der grundlegenden HTML5-Struktur.
*   Erstellung einer navigierbaren Website mit mehreren Seiten.
*   Verwendung von Links zur Verknüpfung von Inhalten.
*   Grundlegende Gestaltung mit Inline CSS im `<head>`-Bereich.
*   Entwicklung eines Verständnisses für die Struktur und Organisation von Webprojekten.

## Aufgabenstellung

Erstellen Sie eine persönliche Homepage zu einem Thema Ihrer Wahl. Das Thema kann aus Ihrem Berufsleben (z.B. ein Portfolio, eine Projektseite) oder Ihrem Privatleben (z.B. ein Hobby, eine Reise, eine Fanseite) stammen. Die Homepage soll aus mindestens drei verlinkten HTML-Seiten bestehen, wobei eine davon die Startseite ist.

### Anforderungen an die Struktur und den Inhalt:

1.  **Startseite (index.html):**
    *   Eine aussagekräftige Überschrift (z.B. `<h1>`).
    *   Eine kurze Einleitung oder Vorstellung zum Thema Ihrer Homepage.
    *   Ein Bild, das zum Thema passt.
    *   Ein **Navigationsbereich** (z.B. mit `<nav>` und `<ul>`/`<li>`-Elementen), der Links zu allen anderen Seiten Ihrer Homepage enthält.

2.  **Weitere Seiten (mindestens 2, z.B. ueber_mich.html, galerie.html, kontakt.html):**
    *   Jede Seite muss einen eindeutigen Titel haben.
    *   Jede Seite muss einen Inhalt aufweisen, der zum Thema der Seite passt (z.B. Text, weitere Bilder, Listen).
    *   Jede Seite muss ebenfalls den **Navigationsbereich** der Startseite enthalten, um eine konsistente Navigation zu gewährleisten.
    *   Stellen Sie sicher, dass alle Seiten untereinander verlinkt sind und man von jeder Seite zu jeder anderen Seite navigieren kann.

### Anforderungen an die Gestaltung (CSS):

*   Die grundlegende Gestaltung (z.B. Schriftart, Textfarbe, Hintergrundfarbe, Abstände) soll für **alle Seiten konsistent** sein.
*   Verwenden Sie **ausschließlich Inline CSS** im `<head>`-Bereich jeder HTML-Datei (innerhalb eines `<style>`-Tags).
*   Experimentieren Sie mit mindestens drei verschiedenen CSS-Eigenschaften (z.B. `color`, `background-color`, `font-family`, `text-align`, `margin`, `padding`).

### Technische Anforderungen:

*   Verwenden Sie die aktuelle HTML5-Dokumentstruktur (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`).
*   Nutzen Sie semantische HTML-Elemente, wo sinnvoll (z.B. `<header>`, `<nav>`, `<main>`, `<footer>`, `<section>`, `<article>`).
*   Stellen Sie sicher, dass Ihr HTML-Code valide ist (optional, aber empfohlen: Nutzung eines HTML-Validators).
*   Benennen Sie Ihre Dateien sinnvoll (z.B. `index.html`, `about.html`, `gallery.html`).

## Abgabekriterien

Ihre Lösung wird anhand der folgenden Punkte bewertet:

*   **Vollständigkeit:** Sind alle geforderten Elemente und Seiten vorhanden?
*   **Funktionalität:** Funktionieren alle Links korrekt? Ist die Navigation intuitiv?
*   **Struktur:** Ist der HTML-Code sauber, semantisch korrekt und gut lesbar?
*   **Gestaltung:** Ist eine grundlegende, konsistente Gestaltung mit Inline CSS vorhanden?
*   **Kreativität:** Wie gut wurde das gewählte Thema umgesetzt und präsentiert?

## Tipps und Hinweise

*   **Planen Sie zuerst:** Überlegen Sie sich, welches Thema Sie wählen und welche Inhalte auf welche Seite kommen sollen.
*   **Struktur vor Inhalt:** Beginnen Sie mit der grundlegenden HTML-Struktur und dem Navigationsbereich, bevor Sie detaillierte Inhalte einfügen.
*   **Konsistenz ist wichtig:** Achten Sie darauf, dass der Navigationsbereich und die grundlegende Gestaltung auf allen Seiten gleich sind.
*   **Testen Sie regelmäßig:** Öffnen Sie Ihre HTML-Dateien im Browser und klicken Sie alle Links durch, um sicherzustellen, dass alles funktioniert.
*   **Fehlerbehebung:** Wenn etwas nicht funktioniert, überprüfen Sie Ihren Code sorgfältig auf Tippfehler oder fehlende Tags.

Viel Erfolg und viel Spaß beim Erstellen Ihrer ersten Homepage!
