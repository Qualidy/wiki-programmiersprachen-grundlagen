# Application Server 1 – Grundlagen

## Lernziele
Nach diesem Kapitel kannst du:
- Einen einfachen **Servlet** erstellen und in einen Application Server deployen.
- HTTP‑Methoden (`GET`, `POST`) unterscheiden.
- Basis‑Sicherheit mit **Basic Auth** realisieren.

---

## Einstieg
Ein Application Server verwaltet Java‑EE‑Komponenten wie Servlets, EJBs und JPA. Für die Demo benutzen wir **Tomcat**.

```java
@WebServlet("/hello")
public class HelloServlet extends HttpServlet {
    protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws IOException {
        resp.setContentType("text/plain");
        resp.getWriter().write("Hello, World!");
    }
}
```

---
