# Low Code 1 – Einführung in Drag‑and‑Drop Tools

## Lernziele
Nach diesem Kapitel kannst du:
- Die Grundprinzipien eines Low‑Code‑Builders beschreiben.
- Einfache Anwendungen mit Drag‑and‑Drop‑Elementen erstellen.
- Datenbindungen zu Datenbanken simulieren.

---

## Einstieg
Low‑Code‑Plattformen wie **Retool**, **Appsmith** oder **Microsoft Power Apps** erlauben das Erstellen von Anwendungen ohne Programmierkenntnisse.

1. **Drag‑and‑Drop**: UI‑Elemente per Maus platzieren.
2. **Verbindungen**: API‑Endpunkte oder Datenbanken an UI‑Elemente binden.
3. **Logik**: Mit visuellen Flow‑Diagrammen Regeln festlegen.

---

## Aufgabe
Erstelle mit Retool einen **Dashboard** für fiktive Verkaufsdaten. Zeige einen **Chart** und eine **Tabelle** an, die aus einer API (`/api/sales`) geladen werden.
