## Einführung

Java ist eine **objektorientierte**, **stark typisierte** Programmiersprache, die 1995 von Sun Microsystems entwickelt wurde. Heute wird Java von Oracle verwaltet.

![Java Logo](https://upload.wikimedia.org/wikipedia/commons/thumb/6/63/Java_logo.svg/300px-Java_logo.svg.png){ align=left width=300px }

**Wichtige Fakten:**
- Java ist **plattformunabhängig** (Write Once, Run Anywhere)
- Java läuft auf der **Java Virtual Machine (JVM)**
- Java wird für Enterprise-Apps, Android-Apps, Web-Services verwendet
- Java ist **stark typisiert** (Typen müssen explizit deklariert werden)

**Anwendungsbereiche:**
- Enterprise-Anwendungen (Spring, Jakarta EE)
- Android-Apps
- Web-Services (REST, SOAP)
- Big Data (Hadoop, Spark)
- Finanzsysteme

## Java-Setup

### Schritt 1: JDK installieren
Lade das **Java Development Kit (JDK)** von [oracle.com/java](https://www.oracle.com/java/technologies/downloads/) herunter.

**Versionen:**
- **LTS (Long Term Support)**: Java 17, Java 21 (empfohlen für Anfänger)
- **Aktuelle Version**: Java 23

### Schritt 2: Installation prüfen
Öffne die Kommandozeile (CMD/Terminal) und führe aus:

```bash
java -version
javac -version
```

Du solltest die Java-Version sehen, z. B.:
```
java version "21.0.1" 2023-10-17 LTS
```

### Schritt 3: IDE installieren (empfohlen)
- **IntelliJ IDEA** (Community Edition - kostenlos)
- **Eclipse**
- **VS Code** mit Java-Erweiterungen

## Erstes Java-Programm

### Schritt 1: Datei erstellen
Erstelle eine Datei namens `HalloWelt.java`:

```java
public class HalloWelt {
    public static void main(String[] args) {
        System.out.println("Hallo, Welt!");
    }
}
```

### Schritt 2: Kompilieren
Im Terminal im selben Verzeichnis wie die Datei:

```bash
javac HalloWelt.java
```

Das erzeugt eine Datei `HalloWelt.class` (Bytecode).

### Schritt 3: Ausführen
```bash
java HalloWelt
```

**Ausgabe:**
```
Hallo, Welt!
```

## Java-Syntax-Regeln

### 1. Klassen
Jede Java-Datei kann **maximal eine public Klasse** enthalten:

```java
public class MeinProgramm {
    // Inhalt der Klasse
}
```

- Dateiname muss mit dem Klassennamen übereinstimmen (Großbuchstabe)
- Java ist **case-sensitive** (Groß-/Kleinschreibung wichtig)

### 2. Methoden
Methoden sind Funktionen in Java:

```java
public class MeinProgramm {
    public static void main(String[] args) {
        // Code hier
    }
}
```

**Wichtige Methoden:**
- `public static void main(String[] args)` - Einstiegspunkt
- `public void meineMethode()` - Eigene Methode
- `public int berechne(int a, int b)` - Methode mit Rückgabewert

### 3. Statements
- Jede Anweisung endet mit einem **Semikolon** `;`
- **Blöcke** werden mit `{ }` umschlossen
- **Einrückungen** sind optional, aber empfohlen (4 Leerzeichen)

```java
public class Beispiel {
    public static void main(String[] args) {
        int x = 10;  // Statement mit Semikolon
        if (x > 5) { // Block beginnt
            System.out.println("x ist größer als 5");
        } // Block endet
    }
}
```

### 4. Kommentare
```java
// Einzeiliger Kommentar

/*
 * Mehrzeiliger Kommentar
 * Zeile 2
 */

/**
 * Dokumentations-Kommentar (für Javadoc)
 */
```

## Datentypen

Java hat zwei Kategorien von Datentypen:

### Primitive Datentypen (8 Typen)

| Typ | Beschreibung | Größe | Wertebereich |
|-----|--------------|-------|--------------|
| `byte` | Ganzzahl | 8 Bit | -128 bis 127 |
| `short` | Ganzzahl | 16 Bit | -32.768 bis 32.767 |
| `int` | Ganzzahl | 32 Bit | -2.147.483.648 bis 2.147.483.647 |
| `long` | Ganzzahl | 64 Bit | -9.223.372.036.854.775.808 bis 9.223.372.036.854.775.807 |
| `float` | Gleitkommazahl | 32 Bit | ±3.4e-38 bis ±3.4e38 |
| `double` | Gleitkommazahl | 64 Bit | ±1.7e-308 bis ±1.7e308 |
| `char` | Zeichen | 16 Bit | Unicode-Zeichen |
| `boolean` | Wahr/Falsch | 1 Bit | `true` oder `false` |

### Referenzdatentypen
- **String** - Text
- **Arrays** - Sammlungen
- **Objekte** - Klassen-Instanzen
- **Collections** - Listen, Sets, Maps

## Variablen deklarieren

### Syntax
```java
Datentyp variableName = wert;
```

### Primitive Datentypen
```java
// Ganzzahlen
byte b = 100;
short s = 1000;
int i = 100000;
long l = 1000000000L; // L am Ende für long

// Gleitkommazahlen
float f = 3.14f; // f am Ende für float
double d = 3.1415926535;

// Zeichen
char c = 'A'; // Einzelzeichen in einfache Anführungszeichen

// Boolean
boolean istAktiv = true;
```

### Referenzdatentypen
```java
// String
String name = "Max Mustermann";

// Array
int[] zahlen = {1, 2, 3, 4, 5};

// Objekt (wird später behandelt)
Person person = new Person();
```

### Konstanten (final)
```java
final double PI = 3.14159;
final int MAX_WERT = 100;

// PI = 3.14; // FEHLER: Konstante kann nicht geändert werden
```

### Namenskonventionen
- **camelCase**: `meineVariable`, `benutzerName`
- **Klassen**: `PascalCase` (Großbuchstabe am Anfang)
- **Konstanten**: `UPPER_CASE` (Großbuchstaben mit Unterstrichen)

```java
// Variablen
int benutzerAlter = 25;
String benutzerName = "Max";

// Klassen
public class MeinProgramm { }

// Konstanten
final double PI = 3.14159;
final int MAX_USERS = 1000;
```

## Operatoren

### Mathematische Operatoren
```java
int a = 10;
int b = 3;

System.out.println(a + b);  // 13
System.out.println(a - b);  // 7
System.out.println(a * b);  // 30
System.out.println(a / b);  // 3 (Ganzzahl-Division)
System.out.println(a % b);  // 1 (Modulo)
```

### Vergleichsoperatoren
```java
int x = 10;
int y = 5;

System.out.println(x == y);  // false
System.out.println(x != y);  // true
System.out.println(x > y);   // true
System.out.println(x < y);   // false
System.out.println(x >= 10); // true
System.out.println(x <= 10); // true
```

### Logische Operatoren
```java
boolean istAdmin = true;
boolean istAktiv = false;

System.out.println(istAdmin && istAktiv); // false (AND)
System.out.println(istAdmin || istAktiv); // true (OR)
System.out.println(!istAdmin);            // false (NOT)
```

### Zuweisungsoperatoren
```java
int x = 10;

x += 5;  // x = x + 5 → 15
x -= 3;  // x = x - 3 → 12
x *= 2;  // x = x * 2 → 24
x /= 4;  // x = x / 4 → 6
x %= 4;  // x = x % 4 → 2
```

## Typ-Konvertierung (Casting)

### Implizite Konvertierung (Widening)
```java
int i = 100;
long l = i; // OK: int → long (kleiner → größer)

float f = 3.14f;
double d = f; // OK: float → double
```

### Explizite Konvertierung (Narrowing)
```java
double d = 3.14;
int i = (int) d; // 3 (Nachkommastellen werden abgeschnitten)

long l = 10000000000L;
int i = (int) l; // Achtung: Überlauf möglich!
```

### String-Konvertierung
```java
// Zahl → String
String text = String.valueOf(42);
String text2 = Integer.toString(42);

// String → Zahl
int zahl = Integer.parseInt("42");
double dezimal = Double.parseDouble("3.14");
```

## Praktische Beispiele

### Beispiel 1: Einfache Berechnung
```java
public class Berechnung {
    public static void main(String[] args) {
        int a = 10;
        int b = 5;
        int summe = a + b;

        System.out.println("Summe: " + summe);
    }
}
```

### Beispiel 2: Temperatur-Umrechnung
```java
public class Temperatur {
    public static void main(String[] args) {
        double celsius = 25.0;
        double fahrenheit = (celsius * 9.0 / 5.0) + 32.0;

        System.out.println(celsius + "°C = " + fahrenheit + "°F");
    }
}
```

### Beispiel 3: Benutzer-Input (mit Scanner)
```java
import java.util.Scanner;

public class BenutzerInput {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Gib deinen Namen ein: ");
        String name = scanner.nextLine();

        System.out.print("Gib dein Alter ein: ");
        int alter = scanner.nextInt();

        System.out.println("Hallo, " + name + "! Du bist " + alter + " Jahre alt.");

        scanner.close();
    }
}
```

## Häufige Fehler & Best Practices

### Fehler 1: Fehlendes Semikolon
```java
// ❌ Falsch
int x = 10

// ✅ Gut
int x = 10;
```

### Fehler 2: Falsche Groß-/Kleinschreibung
```java
// ❌ Falsch (Java ist case-sensitive)
int x = 10;
System.out.println(X); // FEHLER: X nicht definiert

// ✅ Gut
System.out.println(x);
```

### Fehler 3: Falsche Datentypen
```java
// ❌ Falsch (String kann nicht in int konvertiert werden)
int zahl = "42";

// ✅ Gut
int zahl = Integer.parseInt("42");
```

### Fehler 4: Vergessene `L` bei long
```java
// ❌ Falsch (zu groß für int)
long l = 10000000000;

// ✅ Gut
long l = 10000000000L;
```

### Best Practices
1. **Immer `final` für Konstanten** verwenden
2. **Beschreibende Variablennamen** (z. B. `benutzerAlter` statt `a`)
3. **Semikolons** nicht vergessen
4. **Einrückungen** konsistent (4 Leerzeichen)
5. **Kommentare** für komplexe Logik
6. **Try-with-resources** für Scanner (später gelernt)
