# Java Grundlagen 1 – Klassen & Objekte

## Lernziele
Nach diesem Kapitel kannst du:
- Eine Klasse deklarieren und Instanzen erzeugen.
- **Konstruktoren** und **Methoden** nutzen.
- **Encapsulation** durch `private` und `public` verstehen.

---

## Einstieg
Java ist eine objektorientierte Sprache, die auf der JVM läuft.

```java
public class Person {
    private String name;

    public Person(String name) {
        this.name = name;
    }

    public void greet() {
        System.out.println("Hallo, " + name);
    }
}

public class Main {
    public static void main(String[] args) {
        Person p = new Person("Anna");
        p.greet();
    }
}
```
