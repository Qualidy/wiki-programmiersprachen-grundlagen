## Datentypen und Variablen

| Typ | Beispiel | Beschreibung |
|---|---|---|
| `int` | `int age = 25;` | Ganze Zahl |
| `double` | `double price = 19.99;` | Fließkommazahl |
| `boolean` | `boolean isValid = true;` | Wahr/Falsch |
| `String` | `String name = "Alice";` | Text |
| `char` | `char letter = 'A';` | Einzelnes Zeichen |

Kombination aus primitivem Typ und Wrapper‑Klasse (`Integer`, `Double`) erlaubt objektorientierte Nutzung.

## Kontrollstrukturen

### If‑Else

```java
if (age >= 18) {
    System.out.println("Adult");
} else {
    System.out.println("Minor");
}
```

### Switch

```java
switch (day) {
    case MONDAY:
        work();
        break;
    case SATURDAY:
    case SUNDAY:
        relax();
        break;
}
```

### Schleifen

```java
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

while (running) {
    update();
}
```

---

## Ausnahmen behandeln

```java
try {
    int result = divide(a, b);
} catch (ArithmeticException e) {
    System.err.println("Division by zero");
}
```
