## Variablen & Datentypen

| Deklarator | Typ | Beispiel | Beschreibung |
|---|---|---|---|
| `var` | veraltet | `var x = 10;` | Funktionaler Scope, hoisted |
| `let` | block‑scope | `let y = 20;` | moderne Variable |
| `const` | block‑scope | `const PI = 3.14;` | unveränderliche Konstante |

```js
let name = "Ada";
const greeting = `Hello, ${name}!`;
```

## Funktionen

```js
function add(a, b) {
    return a + b;
}

const sum = add(3, 4); // 7
```

Arrow‑Funktionen:

```js
const mul = (a, b) => a * b;
```

## DOM‑Manipulation

```js
// Element auswählen
const btn = document.querySelector('#myBtn');

// Ereignis registrieren
btn.addEventListener('click', () => {
    const p = document.createElement('p');
    p.textContent = 'Button clicked!';
    document.body.appendChild(p);
});
```

## Asynchrone Programmierung

### Promise

```js
function fetchData(url) {
    return new Promise((resolve, reject) => {
        fetch(url)
            .then(response => response.json())
            .then(data => resolve(data))
            .catch(err => reject(err));
    });
}
```

### async/await

```js
async function getInfo() {
    try {
        const data = await fetchData('https://api.example.com');
        console.log(data);
    } catch (err) {
        console.error(err);
    }
}
```
