# JavaScript Grundlagen 1 – Variablen, Datentypen & Operatoren

## Lernziele
Nach diesem Kapitel kannst du:
- **Variablen** mit `var`, `let` und `const` deklarieren.
- Die **Grunddatentypen** (Number, String, Boolean, null, undefined, Symbol, BigInt) beschreiben.
- **Operatoren** wie `+`, `-`, `*`, `/`, `%` und Vergleichsoperatoren nutzen.
- Einfache **if‑Statements** und **switch‑Cases** implementieren.

---

## Einstieg
JavaScript ist die Sprache, die Webbrowser verstehen. Mit ihr kannst du Inhalte dynamisch verändern, Formulare validieren oder Daten aus der API holen.

```javascript
const name = 'Anna';
let alter = 25;

if (alter >= 18) {
  console.log(`${name} ist volljährig.`);
}
```
