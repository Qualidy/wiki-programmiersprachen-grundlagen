# JavaScript Grundlagen 7 – Game of Life Simulation

## Lernziele
Nach diesem Kapitel sollen die Lernenden in der Lage sein:

- HTML‑Strukturen mit `div`‑Elementen aufzubauen
- CSS‑Grid bzw. Flexbox zu nutzen, um ein Raster anzuzeigen
- JavaScript‑Arrays und Schleifen zu verwenden, um das Spiel zu simulieren
- Event‑Handler zu verbinden, damit die Simulation starten/stoppen kann
- Das Ergebnis in der Konsole zu debuggen

## Aufgabe
Erstelle ein interaktives *Game of Life* in HTML, CSS und JavaScript, das in die persönliche Website eingebettet werden kann. Der Fokus liegt darauf, die erlernten Konzepte (Arrays, Schleifen, DOM‑Manipulation, Event‑Handling) zu festigen.

## Konzept

- **Zustand**: 2‑D‑Array `grid[row][col]` mit `true` (lebendig) / `false` (tot)
- **Regeln** (Conway)
+ Jede lebende Zelle mit 2–3 Nachbarn bleibt
+ Jede tote Zelle mit genau 3 Nachbarn wird lebendig
- **Iterieren**: 2 verschachtelte Schleifen über Zeilen und Spalten
- **UI**: Jedes Feld wird als `div` mit `data-row`/`data-col`‑Attribute gerendert

## 1. Grundgerüst
- **HTML**: `<div id="grid"></div>` als Container. Unterhalb einen Button `<button id="toggle">Start / Stop</button>`.
- **CSS**: Grid‑Layout (`display: grid; grid-template-columns: repeat(20, 20px);`). Jede Zelle ist ein `<div class="cell"></div>`.
- **JavaScript**:
  ```js
  const rows = 20, cols = 20;
  let grid = initGrid(rows, cols);
  let interval;
  function initGrid(r, c) { … }
  function step() { … }
  function render() { … }
  document.getElementById("toggle").addEventListener("click", () => {
      if (interval) clearInterval(interval);
      else interval = setInterval(() => { step(); render(); }, 200);
  });
  ```
  * `initGrid` erzeugt ein 2‑D‑Array mit `false`‑Zellen.
  * `step` berechnet den nächsten Zustand nach Conway‑Regeln.
  * `render` aktualisiert die CSS‑Klasse `alive` für jede Zelle.

## 2. Beispiel‑Code (Auszug)
```js
function initGrid(r, c) {
  const g = [];
  for (let i = 0; i < r; i++) {
    const row = [];
    for (let j = 0; j < c; j++) row.push(false);
    g.push(row);
  }
  return g;
}
```

## 3. Einbettung
- Kopiere den HTML‑Block in deine persönliche Website (z. B. `index.html`).
- Füge die CSS‑ und JS‑Dateien (oder die oben genannten Code‑Snippets) ein.
- Teste die Simulation, indem du die Seite im Browser öffnest.

## 4. Aufgabenstellung für die Lernenden
1. Erstelle die Dateien und Code‑Strukturen wie oben beschrieben.
2. Experimentiere mit unterschiedlichen Start‑Zuständen (z. B. *Glider*, *Block*).
3. Dokumentiere in der Markdown‑Datei die Beobachtungen (Was passiert bei welchen Zuständen?).
