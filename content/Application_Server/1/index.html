    Warm-up: Was gehört wohin?

    Ordne die folgenden Begriffe dem richtigen Bereich zu: head oder body.

    • Seitentitel (Tab-Name)
    • sichtbarer Text auf der Seite
    • Verweis auf CSS-Datei
    • Überschrift
    • Meta-Informationen (z. B. charset)
    • Bild auf der Seite

    Schreibe deine Lösung als zwei Listen:

    head:

    • ...

    body:

    • ...
    Lösung

    head:

    • Seitentitel (Tab-Name)
    • Verweis auf CSS-Datei
    • Meta-Informationen (z. B. charset)

    body:

    • sichtbarer Text auf der Seite
    • Überschrift
    • Bild auf der Seite
