    JavaScript Grundlagen 5 – Arrays & Objects¶

    Lernziele

    Nach diesem Kapitel kannst du …

    • Arrays erstellen, manipulieren und durchlaufen.
    • Objekte (Key-Value Paare) verwenden und verändern.
    • Destructuring und Spread Operator anwenden.
    • JSON für Daten austausch nutzen.
    • Map und Set als moderne Datenstrukturen verstehen.

    Einführung¶

    Arrays und Objekte sind die wichtigsten Datenstrukturen in JavaScript: - Arrays: Geordnete Listen von Werten - Objekte: Sammlung von Key-Value Paaren - JSON: Standard für Datenaustausch - Map/Set: Moderne, leistungsfähige Alternativen

  console.log(farben[i]);

  console.log(`${index}: ${farbe}`);

for (const farbe of farben) {
  console.log(farbe);
}

// Map (Transformation)
const gross = farben.map(f => f.toUpperCase());

// Filter
const mitR = farben.filter(f => f.startsWith("R"));
console.log(mitR); // ["Rot"]

// Reduce
const summe = [1, 2, 3, 4].reduce((acc, curr) => acc + curr, 0);
console.log(summe); // 10

// Verschachteltes Destructuring
const user = {
  name: "Max",

};

const { adresse: { stadt, plz } } = user;

map.delete("name");

for (const [key, value] of map) {
  console.log(`${key}: ${value}`);
}

// Konvertierung
const obj = Object.fromEntries(map);
const map2 = new Map(Object.entries(obj));

    Array Methoden Slice, Splice, Map, Filter, Reduce

    Nutze das Array zahlen aus dem vorherigen Beispiel (1, 2, 3, 4, 5). Führe die folgenden Aufgaben aus und zeige die Ergebnisse in der Konsole: 1. Erstelle einen neuen Array teil mit slice, das nur die Werte 2 bis 4 enthält. 2. Verwende splice, um das Element an Index 1 zu entfernen und ersetze es durch die Werte 10 und 11. 3. Erstelle ein neues Array gerade mit map, das jede Zahl um den Faktor 2 multipliziert. 4. Filtere das Array zahlen mit filter, um nur gerade Zahlen zu behalten. 5. Berechne mit reduce die Summe aller Zahlen im Array zahlen.

    const zahlen = [1, 2, 3, 4, 5];
console.log('Ursprünglich:', zahlen);

// 1. slice
const teil = zahlen.slice(1, 4); // Index 1 bis 3
console.log('Slice 1-4:', teil);

// 2. splice
const removed = zahlen.splice(1, 1, 10, 11); // entferne 1 Element, füge 10,11 ein
console.log('Nach splice:', zahlen, 'Removed:', removed);

// 3. map
const gerade = zahlen.map(x => x * 2);
console.log('Map *2:', gerade);

// 4. filter
const geradeZahlen = zahlen.filter(x => x % 2 === 0);
console.log('Filter gerade:', geradeZahlen);

// 5. reduce
const summe = zahlen.reduce((acc, curr) => acc + curr, 0);
console.log('Reduce Summe:', summe);

    JSON (JavaScript Object Notation)¶

    Was ist JSON?¶

    • Textformat für Datenaustausch
    • Basierend auf JavaScript-Objekten
    • Einfach und leichtgewichtig
    • Sprachunabhängig

    Fehler 1: Array-Referenzen¶

    // ❌ Falsch (Referenz-Kopie)
const arr1 = [1, 2, 3];
const arr2 = arr1; // arr2 ist Referenz auf arr1
arr2.push(4);
console.log(arr1); // [1, 2, 3, 4] (arr1 wurde auch geändert!)

// ✅ Gut (echte Kopie)
const arr1 = [1, 2, 3];
const arr2 = [...arr1]; // Spread Operator
arr2.push(4);
console.log(arr1); // [1, 2, 3] (arr1 bleibt unverändert)

    Best Practices¶

    1. Spread Operator für Kopien verwenden
    2. Destructuring für sauberen Code
    3. Optional Chaining für sichere Zugriffe

    5. Map/Set für spezielle Anforderungen
    6. Keine tiefen Verschachtelungen
    7. Konsistente Namenskonventionen
