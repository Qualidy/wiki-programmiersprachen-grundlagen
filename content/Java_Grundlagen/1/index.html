    Java Grundlagen 1 – Setup, Syntax & Datentypen¶

    Lernziele

    Nach diesem Kapitel kannst du …

    • Java auf deinem System installieren und einrichten.
    • Ein erstes Java-Programm schreiben und ausführen.
    • Die Java-Syntax (Klassen, Methoden, Statements) verstehen.
    • Primitive Datentypen und Referenzdatentypen unterscheiden.
    • Variablen deklarieren und initialisieren.

    Einführung¶

    Java ist eine objektorientierte, stark typisierte Programmiersprache, die 1995 von Sun Microsystems entwickelt wurde. Heute wird Java von Oracle verwaltet.

    Wichtige Fakten: - Java ist plattformunabhängig (Write Once, Run Anywhere) - Java läuft auf der Java Virtual Machine (JVM) - Java wird für Enterprise-Apps, Android-Apps, Web-Services verwendet - Java ist stark typisiert (Typen müssen explizit deklariert werden)

    Anwendungsbereiche: - Enterprise-Anwendungen (Spring, Jakarta EE) - Android-Apps - Web-Services (REST, SOAP) - Big Data (Hadoop, Spark) - Finanzsysteme

    Java-Setup¶

    Schritt 1: JDK installieren¶

    Lade das Java Development Kit (JDK) von oracle.com/java herunter.

    Versionen: - LTS (Long Term Support): Java 17, Java 21 (empfohlen für Anfänger) - Aktuelle Version: Java 23

    Schritt 2: Installation prüfen¶

    Öffne die Kommandozeile (CMD/Terminal) und führe aus:

    1
2
    java -version
javac -version

    Du solltest die Java-Version sehen, z. B.: 

    1
    java version "21.0.1" 2023-10-17 LTS

    Schritt 3: IDE installieren (empfohlen)¶

    • IntelliJ IDEA (Community Edition - kostenlos)
    • Eclipse
    • VS Code mit Java-Erweiterungen

    Erstes Java-Programm¶

    Schritt 1: Datei erstellen¶

    Erstelle eine Datei namens HalloWelt.java:

    public static void main(String[] args) {

    Schritt 2: Kompilieren¶

    Im Terminal im selben Verzeichnis wie die Datei:

    1
    javac HalloWelt.java

    Das erzeugt eine Datei HalloWelt.class (Bytecode).

    Schritt 3: Ausführen¶

    1
    java HalloWelt

    Ausgabe: 

    1
    Hallo, Welt!

    Java-Syntax-Regeln¶

    1. Klassen¶

    Jede Java-Datei kann maximal eine public Klasse enthalten:

    1
2
3
    public class MeinProgramm {
    // Inhalt der Klasse
}
    • Dateiname muss mit dem Klassennamen übereinstimmen (Großbuchstabe)
    • Java ist case-sensitive (Groß-/Kleinschreibung wichtig)

    2. Methoden¶

    Methoden sind Funktionen in Java:

    1
2
3
4
5
    public class MeinProgramm {
    public static void main(String[] args) {
        // Code hier
    }
}

    Wichtige Methoden: - public static void main(String[] args) - Einstiegspunkt - public void meineMethode() - Eigene Methode - public int berechne(int a, int b) - Methode mit Rückgabewert

    3. Statements¶

    • Jede Anweisung endet mit einem Semikolon ;
    • Blöcke werden mit { } umschlossen
    • Einrückungen sind optional, aber empfohlen (4 Leerzeichen)
    1
2
3
4
5
6
7
8
    public class Beispiel {
    public static void main(String[] args) {
        int x = 10;  // Statement mit Semikolon
        if (x > 5) { // Block beginnt
            System.out.println("x ist größer als 5");
        } // Block endet
    }
}

    Datentypen¶

    Java hat zwei Kategorien von Datentypen:

    Primitive Datentypen (8 Typen)¶

    Typ Beschreibung Größe Wertebereich
    byte Ganzzahl 8 Bit -128 bis 127
    short Ganzzahl 16 Bit -32.768 bis 32.767
    int Ganzzahl 32 Bit -2.147.483.648 bis 2.147.483.647
    long Ganzzahl 64 Bit -9.223.372.036.854.775.808 bis 9.223.372.036.854.775.807
    float Gleitkommazahl 32 Bit ±3.4e-38 bis ±3.4e38
    double Gleitkommazahl 64 Bit ±1.7e-308 bis ±1.7e308
    char Zeichen 16 Bit Unicode-Zeichen
    boolean Wahr/Falsch 1 Bit true oder false

    Referenzdatentypen¶

    • String - Text
    • Arrays - Sammlungen
    • Objekte - Klassen-Instanzen
    • Collections - Listen, Sets, Maps

    Erstes Java-Programm

    Erstelle eine Java-Klasse HalloJava mit folgenden Anforderungen:

    1. Deklariere eine Konstante PI mit dem Wert 3.14159
    2. Deklariere eine Variable radius mit dem Wert 5.0
    3. Berechne den Flächeninhalt eines Kreises: Fläche = PI * radius²
    4. Gib das Ergebnis in der Konsole aus mit dem Format: "Fläche: [wert]"

    Bonus: Verwende double für alle Berechnungen und gib das Ergebnis mit 2 Nachkommastellen aus.

    Lösung 
     1

 4

    public class HalloJava {
    public static void main(String[] args) {
        // Konstante
        final double PI = 3.14159;

        // Variable
        double radius = 5.0;

        // Berechnung
        double flaeche = PI * radius * radius;

        // Ausgabe
        System.out.println("Fläche: " + flaeche);

        // Bonus: Mit 2 Nachkommastellen
        System.out.printf("Fläche: %.2f%n", flaeche);
    }
}

    Häufige Fehler & Best Practices¶

    Fehler 1: Fehlendes Semikolon¶

    1
2
3
4
5
    // ❌ Falsch
int x = 10

// ✅ Gut
int x = 10;

    Fehler 2: Falsche Groß-/Kleinschreibung¶

    1
2
3
4
5
6
    // ❌ Falsch (Java ist case-sensitive)
int x = 10;
System.out.println(X); // FEHLER: X nicht definiert

// ✅ Gut
System.out.println(x);

    Fehler 3: Falsche Datentypen¶

    1
2
3
4
5
    // ❌ Falsch (String kann nicht in int konvertiert werden)
int zahl = "42";

// ✅ Gut
int zahl = Integer.parseInt("42");

    Fehler 4: Vergessene L bei long¶

    1
2
3
4
5
    // ❌ Falsch (zu groß für int)
long l = 10000000000;

// ✅ Gut
long l = 10000000000L;

    Best Practices¶

    1. Immer final für Konstanten verwenden
    2. Beschreibende Variablennamen (z. B. benutzerAlter statt a)
    3. Semikolons nicht vergessen
    4. Einrückungen konsistent (4 Leerzeichen)
    5. Kommentare für komplexe Logik
    6. Try-with-resources für Scanner (später gelernt)
