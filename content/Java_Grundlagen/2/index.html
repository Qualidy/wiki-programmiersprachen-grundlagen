    Java Grundlagen 2 – Kontrollstrukturen & Schleifen¶

    Lernziele

    Nach diesem Kapitel kannst du …

    • Kontrollstrukturen (if/else, switch) für Entscheidungen nutzen.
    • Schleifen (for, while, do-while) für Wiederholungen verwenden.
    • Break und Continue für Schleifensteuerung.
    • Verschachtelte Schleifen verstehen und anwenden.
    • Methode mit Kontrollstrukturen schreiben.

    Einführung¶

    Kontrollstrukturen und Schleifen sind die Grundbausteine jeder Programmierung. Sie ermöglichen: - Logische Entscheidungen (if/else, switch) - Wiederholungen (for, while) - Flexible Programmflüsse - Effiziente Datenverarbeitung

    Schreibe eine Funktion bewertung(int punkte), die: 1. Punkte in Note umwandelt (0-100) 2. Verwende switch für die Logik 3. Gib die Note als String zurück

    public class Bewertung {
    // Mit Switch
    public static String bewertungSwitch(int punkte) {
        if (punkte < 0 || punkte > 100) {
            return "Ungültige Punktzahl";
        }

        switch (punkte / 10) {
            case 10:
            case 9:
                return "Sehr gut";
            case 8:
            case 7:
                return "Gut";
            case 6:
            case 5:
                return "Befriedigend";
            default:
                return "Nicht bestanden";
        }
    }

    // Mit Ternary Operator
    public static String bewertungTernary(int punkte) {
        if (punkte < 0 || punkte > 100) {
            return "Ungültige Punktzahl";
        }

        return punkte >= 90 ? "Sehr gut" :
               punkte >= 75 ? "Gut" :
               punkte >= 50 ? "Befriedigend" :
               "Nicht bestanden";
    }

    public static void main(String[] args) {
        // Test
        System.out.println("95 Punkte: " + bewertungSwitch(95)); // Sehr gut
        System.out.println("80 Punkte: " + bewertungSwitch(80)); // Gut
        System.out.println("60 Punkte: " + bewertungSwitch(60)); // Befriedigend
        System.out.println("30 Punkte: " + bewertungSwitch(30)); // Nicht bestanden

        // Bonus
        System.out.println("95 Punkte (Ternary): " + bewertungTernary(95));
    }
}

for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break; // Stoppt bei 5
    }
    System.out.println(i);
}

// Continue: Überspringt aktuelle Iteration
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
        continue; // Überspringt gerade Zahlen
    }
    System.out.println(i); // Nur ungerade Zahlen
}

    public class Statistik {
    // Mit for-Schleife
    public static String statistik(int[] zahlen) {
        if (zahlen == null || zahlen.length == 0) {
            return "Keine Zahlen vorhanden";
        }

        int summe = 0;
        int min = zahlen[0];
        int max = zahlen[0];
        int geradeCount = 0;

        for (int i = 0; i < zahlen.length; i++) {
            int zahl = zahlen[i];
            summe += zahl;

            if (zahl < min) min = zahl;
            if (zahl > max) max = zahl;
            if (zahl % 2 == 0) geradeCount++;
        }

        double durchschnitt = (double) summe / zahlen.length;

        return String.format(
            "Summe: %d, Durchschnitt: %.2f, Min: %d, Max: %d, Gerade: %d",
            summe, durchschnitt, min, max, geradeCount
        );
    }

    // Mit for-each
    public static String statistikForEach(int[] zahlen) {
        if (zahlen == null || zahlen.length == 0) {
            return "Keine Zahlen vorhanden";
        }

        int summe = 0;
        int min = Integer.MAX_VALUE;
        int max = Integer.MIN_VALUE;
        int geradeCount = 0;

        for (int zahl : zahlen) {
            summe += zahl;
            min = Math.min(min, zahl);
            max = Math.max(max, zahl);
            if (zahl % 2 == 0) geradeCount++;
        }

        double durchschnitt = (double) summe / zahlen.length;

        return String.format(
            "Summe: %d, Durchschnitt: %.2f, Min: %d, Max: %d, Gerade: %d",
            summe, durchschnitt, min, max, geradeCount
        );
    }

    // Mit Streams (Bonus)
    public static String statistikStream(int[] zahlen) {
        if (zahlen == null || zahlen.length == 0) {
            return "Keine Zahlen vorhanden";
        }

        int summe = java.util.Arrays.stream(zahlen).sum();
        int min = java.util.Arrays.stream(zahlen).min().getAsInt();
        int max = java.util.Arrays.stream(zahlen).max().getAsInt();
        long geradeCount = java.util.Arrays.stream(zahlen)
            .filter(z -> z % 2 == 0)
            .count();

        double durchschnitt = (double) summe / zahlen.length;

        return String.format(
            "Summe: %d, Durchschnitt: %.2f, Min: %d, Max: %d, Gerade: %d",
            summe, durchschnitt, min, max, geradeCount
        );
    }

    public static void main(String[] args) {
        int[] zahlen = {10, 25, 30, 45, 50, 65, 70, 85, 90, 100};

        System.out.println("Mit for-Schleife:");
        System.out.println(statistik(zahlen));

        System.out.println("\nMit for-each:");
        System.out.println(statistikForEach(zahlen));

        System.out.println("\nMit Streams:");
        System.out.println(statistikStream(zahlen));
    }
}

    import java.util.Scanner;

public class ZahlenRaten {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int ziel = 42;
        int versuch;
        int anzahl = 0;

        do {
            System.out.print("Gib eine Zahl ein: ");
            versuch = scanner.nextInt();
            anzahl++;

            if (versuch < ziel) {
                System.out.println("Zu klein!");
            } else if (versuch > ziel) {
                System.out.println("Zu groß!");
            } else {
                System.out.println("Richtig! Versuche: " + anzahl);
            }
        } while (versuch != ziel);

        scanner.close();
    }
}

    Best Practices¶

    1. Klammern immer verwenden bei if/else, auch bei Einzeiler
    2. For-Each für Arrays/Listen statt Index-Schleifen
    3. Switch für viele Cases statt langer if-else-Ketten
    4. Break/Continue sparsam verwenden
    5. Keine unendlichen Schleifen ohne klaren Exit
    6. Komplexe Bedingungen in Variablen auslagern

    Zusammenfassung & Checkliste¶

    Wichtige Konzepte¶

    • ✅ If/Else: Bedingte Ausführung
    • ✅ Switch: Mehrfach-Verzweigung
    • ✅ Ternary: Kurzform für if-else
    • ✅ For-Schleife: Zählschleife
    • ✅ While-Schleife: Bedingungsschleife
    • ✅ Do-While: Schleife mit mindestens einer Iteration
    • ✅ Break/Continue: Schleifensteuerung

    Checkliste für Code¶

    • [ ] Klammern bei Kontrollstrukturen
    • [ ] Break in jedem switch-case
    • [ ] For-Each für Arrays/Listen
    • [ ] Keine unendlichen Schleifen
    • [ ] Bedingungen verständlich
    • [ ] Komplexe Logik kommentiert
